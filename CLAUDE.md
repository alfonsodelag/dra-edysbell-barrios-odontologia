# CLAUDE.md

## Project Context

This project is a website for:

**Dra. Edysbell Barrios Odontóloga**

The site is being built from a previous dental website template originally created for Odonto Panamá. The goal is to convert the project into a new, independent, polished website for Dra. Edysbell Barrios without breaking the existing technical structure.

The site should feel like a modern boutique dental practice: clean, trustworthy, warm, professional, aesthetic, and human.

## Primary Objective

Transform the existing project into a complete website for **Dra. Edysbell Barrios Odontóloga**, focused on:

- Odontología estética
- Salud oral
- Atención personalizada
- Diseño y armonía de sonrisa
- Limpieza, prevención y cuidado dental
- Confianza, profesionalismo and modern dental care

Do not treat this as a generic dental site. It should feel personalized, elegant, premium, and credible.

## Color Palette

Use this color system:

```css
--primary: #1e6f8c;
--accent: #20afc0;
--dark: #173246;
--cream: #f7f2ea;
--text: #1b1b1b;
--white: #ffffff;
```

### Usage Rules

- Use `#1E6F8C` as the main brand color.
- Use `#20AFC0` only as an accent for small details, highlights, icons, hover states, or secondary visual emphasis.
- Use `#173246` for strong headings, dark sections, footer backgrounds, or elegant contrast.
- Use `#F7F2EA` for soft background sections.
- Use `#1B1B1B` for main text.
- Avoid using too much bright turquoise. It should accent the brand, not dominate it.
